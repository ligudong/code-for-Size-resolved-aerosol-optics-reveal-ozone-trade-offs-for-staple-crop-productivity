# code-for-Size-resolved aerosol optics reveal ozone trade-offs for staple crop productivity

Project Overview

This project integrates size-resolved aerosol optical depth (fAOD and cAOD), high-resolution ozone (AOT40), satellite solar-induced chlorophyll fluorescence (SIF), crop phenology, yield, management, and climate data for maize, rice, and wheat in China. We build scenario simulations and spatial fixed-effect models to quantify the compound effects of aerosols and ozone on crop photosynthesis and yield. The codebase covers data processing, construction of spatial panel data, regression analysis, scenario simulations, robustness checks, and the generation and export of main and supplementary figures.

Directory Structure and File Description

The project consists of four main components: R scripts, data, and output figures.

R folder contains all scripts required for the analyses. Scripts are organized by prefix:
00_ – loading required R packages and font configuration (00_load_packages.R), helper functions for data processing and plotting (00_helper_functions.R), and model/formula definitions (00_formula_definitions.R).
01_prepare_ – scripts for preprocessing raw data.
02_ – scripts for generating intermediate data used in analyses.
03_plot_ – scripts for producing main and supplementary figures. Prefix 03_plot_figure_ generates main text figures, while 03_plot_figureS_ produces supplementary figures. The suffix number corresponds to the figure number. Detailed intermediate computations are embedded within each figure-generating script.

data folder contains all datasets used in the analyses, organized into two subfolders:
raw - raw, unprocessed datasets including publicly available remote sensing, air pollution, crop yield, and phenology data. Download links are provided in Table S12 of the article. Some large or restricted datasets are not included.
derived – intermediate processed datasets and outputs generated from analyses.
Note: Raw data are not included in this repository due to size and licensing constraints. To obtain the data, contact: ligudong@mail.bnu.edu.cn.

figures folder stores all output figures and tables generated by the scripts.
main – figures presented in the main text.
supplementary – supplementary figures and tables.

All figures are provided in the paper and supplementary materials.
